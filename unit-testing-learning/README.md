# Unit Testing Playground

**Scope:**  
A lightweight Java project created for learning purposes, focused on practicing unit testing using JUnit and Mockito. 
It includes simple and illustrative tests.

- 📦 **Gradle 8.10**
- ✅ **JUnit 5**
